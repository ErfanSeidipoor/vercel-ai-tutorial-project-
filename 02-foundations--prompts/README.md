# foundations prompts

This project is based on the [foundations prompts](https://sdk.vercel.ai/docs/foundations/prompts)

This tutorial project is designed to help you learn the fundamentals of AI prompts and how to create effective conversational interfaces.

## Project Structure

---

The project is structured into several folders, each containing a specific script:

- `src/1-basic.ts`: Basic prompt example
- `src/2-dynamic-data.ts`: Dynamic data prompt example
- `src/3-system-prompts.ts`: System messages prompt example
- `src/4-message-prompts.ts`: Message prompts example
- `src/5-user-message-binary-image.ts`: User message binary image example
- `src/6-base-64-image.ts`: Base 64 image example
- `src/7-image-url.ts`: Image URL example
- `src/8-pdf.ts`: PDF example
- `src/9-mp3-audio.ts`: MP3 audio example
- `src/10-assistant-messages.ts`: Assistant messages example
- `src/11-assistant-message-with-tool-call.ts`: Assistant message with tool call example
- `src/12-multi-modal-tool-results.ts`: Multi-modal tool results example
- `src/13-system-messages.ts`: System messages example
