# guides rag-chatbot

This project is based on the [guides rag-chatbot](https://sdk.vercel.ai/docs/guides/rag-chatbot#update-server-action)

This project is a tutorial guide to building a chatbot using Next.js and
Drizzle-ORM. It covers the basics of setting up a new project, integrating
with the OpenAI API, and creating tools to enhance the chatbot's
functionality.
